# E-commerce Conversion Funnel Dashboard

## Project Overview
This project analyzes an e-commerce conversion funnel using SQL and Power BI. The dashboard provides insights into customer behavior, revenue performance, traffic sources, and conversion rates to support business decision-making.

## Tools Used
- MySQL
- SQL
- Power BI

## Key Performance Indicators (KPIs)
- Total Revenue: $212.9K
- Total Orders: 818
- Average Order Value: $260.8
- Overall Conversion Rate: 16.4%

## Dashboard Features
- Revenue by Traffic Source
- Conversion Rate by Traffic Source
- Weekly Cohort Conversion Trend
- Device-wise Funnel Analysis
- Interactive filters for Traffic Source and Device

## Key Insights
- Organic traffic generated the highest revenue.
- Social traffic achieved the highest conversion rate.
- Mobile users accounted for the largest share of customer activity.
- Weekly conversion rates changed over time, highlighting opportunities to improve marketing performance.
